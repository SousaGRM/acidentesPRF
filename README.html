Introduo

Este documento consta da anlise exploratria de dados inicial e visualizao dos dados de acidentes nas rodovias federais brasileiras (aka BR). Os dados foram coletados no site oficial da Polcia Rodoviria Federal do Brasil (PRF). A anlise foi realizada utilizando-se a linguagem R.

Licena

Dados Abertos so dados institucionais, disponibilizados em formato legvel por mquina e sem restrio de licenas, patentes ou mecanismos de controle, que qualquer pessoa pode livremente us-los, reutiliz-los e redistribu-los. Os dados classificados como abertos podem ser utilizados de vrias formas, seja pelo prprio governo ou pela sociedade, como, por exemplo, no desenvolvimento de aplicativos, que exibem informaes de forma grfica e interativa. Trecho extrado do site oficial.

Objetivos

  • Analisar os dados abertos afim de encontrar possveis padres nos acidentes ocorridos nas BRs;

  • Desenvolver visualizaes que facilitem o entedimento da anlise.

Preparando e Combinando as Tabelas

A tabela de 2017 foi filtrada para que os dados no ficassem desproporcionais, ou seja, janeiro, fevereiro, maro e abril teriam seis tabelas, enquanto que os restantes teriam uma tabela a menos, visto que na tabela de 2022 ainda no constam os meses que viro.

Tambm havia uma inconsistncia no tipo de dado da coluna latitude, foi necessrio igualar os tipos para poder combinar as tabelas.

# Preparing to Combine All Datasets
datatran2017_f <- filter(datatran2017, month(datatran2017$data_inversa) > 4)

datatran2021$latitude <- as.double(datatran2021$latitude)

all_datatran <- bind_rows(datatran2017_f, datatran2018, datatran2019,

Formatando e Organizando Colunas

Foram criadas as colunas de mes e ano extraindo da coluna data_inversae realocadas para prximo da data

# Formatting and Organizing Columns
all_datatran$mes <- month(all_datatran$data_inversa,
                          label = TRUE, abbr = TRUE)

all_datatran$ano <- year(all_datatran$data_inversa)

all_datatran <- all_datatran %>%
  relocate(mes, .after = data_inversa) %>%
  relocate(ano, .after = data_inversa)

Visualizao 01

Na visualizao abaixo possvel observar uma maior mdia de acidentes no mes de dezembro, isso pode ter ocorrido devido as festas de final de ano (natal e rveillon) onde geralmente as famlias viajam para se reunir com os familiares e se divertir com amigos aumentando assim o nmero de veculos circulando e a probabilidade de acidentes.

# Visualization 01
ggplot(acidentes_mes, aes(x = mes, y = media)) +
  geom_segment(aes(x = mes, xend = mes, y = 0, yend = media),
               color = viridis(12), size = 1) +
  geom_point(size = 4, color = viridis(12)) +
  coord_flip() +
  annotate("text", x = 12, y = 6648 + 500, label = "6648",
           color = "black", size = 4, fontface = "bold") +
  labs(title = "Qual foi o ms com a maior mdia de acidentes?",
       x = "", y = "", caption = paste(
         "Fonte: Polcia Rodoviria Federal, Brasil", "\n",
         "Perodo: mai/2017 a abr/2022", "\n",
         "ltima atualizao:", today())) +
  theme_classic() +
  theme(axis.text.y = element_text(size = 15, face = "bold"),
        plot.subtitle = element_text(size = 8),
        plot.caption = element_text(size = 5))


Visualizao 02

J a visualizao seguinte mostra a evoluo no nmero de acidentes durante os meses e agrupados por ano. Aqui possvel perceber claramente o que foi visto no grfico anterior, uma alta de acidentes ao chegar no ms de dezembro, tendo o dezembro de 2017 como o pico de acidentes.

Aps o ano de 2017 observa-se uma reduo no nmero de acidentes embora ainda visto a alta de dezembro em cada um dos anos seguintes.

Outro ponto a sinalizar no grfico a baixa de acidentes no ms de abril de 2020, isso provavelmente aconteceu por ter sido o incio das medidas de isolamento devido a pandemia da Covid19 no Brasil.

Visualizao 03

Para finalizar, o prximo grfico mostra a distribuio no nmero de acidentes entre as unidades federativas do Brasil. possvel perceber dois grandes grupos por nmero de acidentes. O primeiro grupo liderado pelo estado de Minas Gerais, e que conta com Santa Catarina e Paran se encontram com os maiores nmeros de acidentes na faixa dos 40 mil. J o segundo grupo conta com o Rio Grande do Sul, Rio de Janeiro e So Paulo com cerca de 20 mil acidentes.

Como foi observado neste grfico, os estados da regio sul e sudeste do Brasil (com exceo do Espirito Santo) brigam pelo pdio de estados com mais acidentes em rodovias federais, visto que so as regies mais populosas do pas.

Concluso

Diante do que foi encontrado, necessrio que sejam desenvolvidos programas de preveno, sobretudo nas regies sul e sudeste do Brasil, anteriormente ao ms de dezembro, afim de educar a populao e reduzir o nmero de acidentes.

Em conjunto com esses programas e aliado a equipe de dados da PRF, pode ser possvel a criao de um sistema de recompensas para bons condutores, podendo ganhar descontos no IPVA, por exemplo.

O contnuo mapeamento dos acidentes nas rodovias indispensvel para que a PRF possa continuar garantindo um bom atendimento em casos de acidente.
